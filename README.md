# XPath Helper - Chrome 扩展

一个功能强大的Chrome扩展，帮助你快速测试和验证XPath表达式。

## ✨ 功能特性

- 🎯 **侧边栏模式**：持久显示，不会像popup那样点击外部就消失
- 🔍 **XPath查询**：输入任意XPath表达式，实时查询页面元素
- 📋 **复制功能**：支持单个复制和批量复制所有匹配结果
- 🌟 **元素高亮**：点击高亮按钮，在页面中直观显示匹配的元素（5秒动画）
- ⚡ **快速示例**：内置常用XPath模板，一键快速测试
- 💎 **现代UI**：美观的渐变设计，流畅的交互体验
- 📝 **历史记录**：自动保存查询历史（最近20条）
- 🎨 **语法高亮**：结果展示清晰，区分元素类型

## 安装方法

### 开发者模式安装

1. 打开Chrome浏览器，访问 `chrome://extensions/`
2. 在右上角启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目的文件夹
5. 扩展安装成功！你会在浏览器工具栏看到扩展图标

## 使用方法

### 打开侧边栏

1. **方式一**：点击浏览器工具栏中的扩展图标
2. **方式二**：右键点击扩展图标 → 选择"打开侧面板"
3. 侧边栏会在浏览器右侧打开，保持显示不会消失

### 查询XPath

1. **输入XPath**：在输入框中输入XPath表达式
   - 例如：`//div[@class='content']`
   - 例如：`//a[@href]/@href`
   - 例如：`//h1/text()`
2. **执行查询**：点击"🔎 查询"按钮或按 `Ctrl+Enter` (Mac: `Cmd+Enter`)
3. **快速示例**：点击上方的快速示例按钮，一键测试常用表达式

### 操作结果

- **📋 复制**：点击单个结果的"复制"按钮
- **📋 复制全部**：批量复制所有匹配结果
- **🎯 高亮**：在页面中高亮显示该元素（5秒动画效果）
- **🗑️ 清除**：清空当前结果和输入

## XPath表达式示例

```xpath
// 选择所有div元素
//div

// 选择特定class的元素
//*[@class='content']

// 选择包含特定文本的元素
//*[contains(text(), '搜索')]

// 选择链接的href属性
//a/@href

// 选择第一个段落
//p[1]

// 选择所有文本节点
//text()

// 组合条件
//div[@class='item' and @data-id]
```

## 技术栈

- **Manifest V3**：使用最新的Chrome扩展API
- **原生JavaScript**：无依赖，性能优秀
- **现代CSS**：渐变、动画、响应式设计

## 键盘快捷键

- `Ctrl+Enter` / `Cmd+Enter`：执行查询
- 输入框支持多行输入，方便编写复杂的XPath表达式

## 注意事项

1. **侧边栏模式**：扩展以侧边栏形式显示，不会因为点击外部而消失
2. **权限要求**：需要访问当前标签页以执行XPath查询
3. **安全限制**：某些网站（如chrome://、about://等）受浏览器保护，无法查询
4. **大小写敏感**：XPath表达式区分大小写
5. **高亮效果**：高亮动画持续5秒后自动消失
6. **历史记录**：自动保存最近20条查询记录
